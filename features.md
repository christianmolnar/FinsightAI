# Main Features

## Dashboard
- Portfolio Value
- Profits Made
- Losses
- Net Position
- Filters: Day, Month, Year, Custom

## Recommendations
- Top 10 symbols from Dow Jones, Nasdaq, Futures
- Indicators: RSI, MACD, Moving Average
- Action Button: Buy

## Tracking Table
- Date Purchased
- Purchase Price
- Current Price
- $ and % Gain/Loss
- Recommendation: Hold/Sell

## Transactions
- Buy More: specify shares/dollars
- Sell: moves record to Historical Table
