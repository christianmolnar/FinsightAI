# Prediction Models

## Modeling Strategy
- Use historical data to forecast movement and volatility

## Models
1. XGBoost (classification)
2. Facebook Prophet (trend forecasting)
3. RSI, MACD, SMA (technical indicators)
4. GARCH (volatility)

## Implementation
- Offline training
- Store model artifacts
- Serve predictions via REST API
- Trigger retraining with CI/CD
