# FInsightAI Learning Triggers
*Automated Learning and Improvement Catalysts*

## Performance-Based Learning Triggers

### Trade Outcome Analysis
**Trigger Condition**: After each completed trade (win or loss)
**Learning Actions**:
- [ ] **Pattern Recognition Update**: Analyze setup patterns that led to trade
- [ ] **Strategy Effectiveness**: Update strategy success/failure statistics
- [ ] **Market Condition Correlation**: Correlate outcome with market conditions
- [ ] **Timing Analysis**: Evaluate entry and exit timing effectiveness
- [ ] **Risk Assessment Validation**: Compare actual vs. predicted risk

**Memory Updates**:
- Update episodic memory with trade details and context
- Enhance semantic memory with pattern recognition insights
- Refine procedural memory with improved decision processes

### Portfolio Performance Milestones
**Trigger Conditions**:
- Weekly performance review completion
- Monthly portfolio assessment
- Quarterly comprehensive analysis
- Annual strategy evaluation

**Learning Actions**:
- [ ] **Attribution Analysis**: Deep dive into performance attribution
- [ ] **Strategy Comparison**: Compare multiple strategies' effectiveness
- [ ] **Risk-Adjusted Performance**: Calculate and trend risk-adjusted returns
- [ ] **Benchmark Analysis**: Analyze performance vs. relevant benchmarks
- [ ] **Client Satisfaction**: Correlate performance with client feedback

## Market Event Learning Triggers

### Volatility Spike Response
**Trigger Condition**: VIX increases >20% in single day OR portfolio loses >3% in day
**Learning Actions**:
- [ ] **Crisis Response Analysis**: Evaluate how well crisis protocols worked
- [ ] **Correlation Breakdown**: Analyze unexpected correlation changes
- [ ] **Liquidity Assessment**: Review liquidity assumptions under stress
- [ ] **Strategy Resilience**: Test strategy performance in volatile conditions
- [ ] **Recovery Pattern**: Document market recovery patterns and positioning

### Economic Event Impact
**Trigger Conditions**:

- Federal Reserve announcements and policy changes
- Geopolitical events with market impact
- Sector-specific disruption events

**Learning Actions**:
- [ ] **Event Response Pattern**: Analyze market response patterns to similar events
- [ ] **Positioning Effectiveness**: Evaluate pre-event positioning decisions
- [ ] **Reaction Speed**: Assess speed and accuracy of response
- [ ] **Information Integration**: Improve event information processing
- [ ] **Prediction Accuracy**: Validate and improve event impact predictions
